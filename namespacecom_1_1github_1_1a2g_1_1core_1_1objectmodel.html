Detailed Description

Infact, the panel classes that exist in the obejctmodel folder are for GWT. If the runtime is set to use the SwingFactory then the runtime will use the panels in core.swing.panel instead.

All the classes in com.github.a2g.core.objectmodel are represented here.
Understand this, and your half way to understanding the a2g library.
