Detailed Description

The ChoicesPanel forms the main part of the MasterPresenter.

It's where the dialog choices are listed during conversations. You won't find a reference to the Choices class here however. The ChoicesPresenter holds the Choices object and updates the ChoicesPanel occasionally.
