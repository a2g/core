Detailed Description

The authoredroom package contains classes used to author a room.

An authored room must extend RoomBase and implement IAmARoom.

It's implementation of onReceiveGameAPIObject must call 'super.setApi(api)' with the IAmTheMainAPI interface it recieves.
A basic authored room might start out like this: 

public class MyAuthoredRoom extends RoomBase implements  IAmARoom
{
    public MyAuthoredRoom(){
        super();
    }   
    
    @Override
    public void onReceiveGameAPIObject(IAmTheMainAPI api)
    {
        super.setApi( api);
    }
    
    @Override
    public void onLoadResources(){
    }

    @Override
    public void onPrepareRoomForFocus(){
    }
    
    @Override
    public void onEveryFrame(){
    }

    @Override
    public BaseAction onEnterRoom(){
        return doNothing();
    }
    
    @Override
    public BaseAction onCommandLineExecute(int verb, SentenceUnit objA, SentenceUnit objB,  double x, double y){        
        return doNothing();
    }
    
    @Override
    public ChoicesActionBase onChoice(int i) {
        return null;
    }
}
