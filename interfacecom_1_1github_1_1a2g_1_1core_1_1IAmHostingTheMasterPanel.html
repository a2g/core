Is there to persist state data at the time the state data change occurs.

The setting of most types of state data is ultimately done by this method. It is needed for the inventory to remember it contains an object when moving from one room to the next. It is also called in reponse to calls of com.github.a2g.core.authoredroom.IAmTheMainAPI.setValue(String name, int value);
