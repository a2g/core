static checking

Strict static checking is useful for adventure game scripts because adventure games typically allow for the construction of hundreds or thousands of of verb-noun combinations per scene. It helps to have allow static checking check for as many errors in the handling of these combinations as is possible, because the more combinations that are individually handled, the more immersive, and enjoyable the game experience is.
