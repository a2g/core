Detailed Description

A2g expects the InventoryItemCollection and RoomObjectCollection to be populated by two methods on the Api (see below).

It expects these calls to be made during the IAmARoom.onLoadResources() method of a room.

These calls can either be made:

  • Manually - use the documentation below
  • Automatically via code generation - see CodeGenerator
